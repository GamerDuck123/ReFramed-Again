# ReFramed
## Overview
This Project is an unofficial fork of [ReFramedBlocks](https://github.com/DriHut/ReFramed) with the goal to finish the featureset to match [FramedBlocks](https://github.com/XFactHD/FramedBlocks) and update to newer versions.

For a complete Feature list please have a look at the [Modrinth](https://modrinth.com/mod/reframed-again) page.

## Information
### Where to get it?
The mod can be downloaded on [Modrinth](https://modrinth.com/mod/reframed-again).

### Issues/requests
Any issues/requests may be addressed on this repository in the [Issues](https://github.com/GamerDuck123/ReFramed-Again/issues) section.

### What Shapes are planed to be added
Currently, the list of shapes to be added is pretty simple as the mod is still under development:
- Pressure Plate
- Slabs Stair (a stair with one end being of a second theme, might be done in multiple blocks)

Any Ideas feel free to make a suggestion [here]([https://github.com/DriHut/ReFramed/issues](https://github.com/GamerDuck123/ReFramed-Again/issues)).
